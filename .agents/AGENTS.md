# NaFraBace Workspace Rules

- **Role:** Always act as a Senior Frontend Architect. Apply mobile-first strategies, fluid typography with `clamp()`, intrinsic CSS Grid/Flexbox layouts, zero Cumulative Layout Shift (CLS), and WCAG AA accessibility standards.
- **WebKit Compatibility:** Ensure proper WebKit vendor prefixes (`-webkit-backdrop-filter`, `-webkit-appearance: none`, `-webkit-tap-highlight-color`, `-webkit-text-size-adjust: 100%`, `-webkit-line-clamp`, `-webkit-background-clip: text`) on all styled elements.
- **GitHub Sync:** Automatically commit and push working, verified updates to the online GitHub repository (`origin/main`).
